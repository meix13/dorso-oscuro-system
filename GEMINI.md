# Dorso Oscuro - Sistema para FoundryVTT

## Descripción del Proyecto
Dorso Oscuro es un juego de rol propio que integra un sistema de cartas. Este proyecto consiste en la implementación de las mecánicas de este juego como un sistema (system) para la plataforma Foundry Virtual Tabletop (FoundryVTT).

## Tecnologías y Versiones
- **Plataforma:** FoundryVTT
- **Versión de API objetivo:** V14
